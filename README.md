# GIS Portfolio

This portfolio is a static site designed for sharing GIS work with hiring managers.

## Add your files

1. Put project screenshots in `assets/screenshots/`
2. Put downloadable project files in `assets/projects/`
3. Update `projects.js` with:
   - project title
   - summary
   - screenshot path
   - tools and skills
   - one or more downloadable files
4. Replace the contact placeholders in `index.html`

## Notes

- Project files are linked as downloads, not embedded in the page.
- If a screenshot is missing, the card will show a placeholder.
- You can duplicate a project object in `projects.js` to add more projects.
